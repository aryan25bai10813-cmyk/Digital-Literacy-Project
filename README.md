Project Overview



This project is aimed at enhancing digital literacy skills that are essential in today’s technology-driven world. It covers key areas such as responsible internet usage, professional email communication, use of digital platforms, and awareness of cyber threats.



In addition to theoretical understanding, the project also includes hands-on activities like designing a presentation, creating an online portfolio, working with digital tools, and analyzing cybersecurity risks.
